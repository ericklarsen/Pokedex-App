## Development Pokedex-App

### Clone the repository

```sh
git@github.com:ericklarsen/Pokedex-App.git
cd Pokedex-App
```

### Install dependency

```sh
# use npm
npm install
```

### Running dev server on local machine

```sh
npm run dev
```
